VisPy documentation
===================

VisPy is the current stable Python library for high-performance,
interactive 2D and 3D scientific visualization. It uses OpenGL to display
large datasets and integrates with Python GUI and notebook environments.

VisPy 2 is a separate, experimental architecture under development. Existing
VisPy applications and documentation routes continue to work.
