Classic VisPy
=============

Classic VisPy is the established Python library for high-performance,
interactive 2D and 3D scientific visualization. It uses OpenGL to display
large datasets and integrates with Python GUI and notebook environments.

Classic VisPy remains available while the experimental VisPy 2 architecture
develops. Existing applications and documentation routes continue to work.
